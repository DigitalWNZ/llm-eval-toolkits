READY TO USE
------------
This is a complete, self-contained toolkit. Everything needed to run
performance tests on Gemini models is included.

QUICK START
-----------
1. Read: README.md
2. Setup: bash setup.sh (on Debian 12 VM)
3. Run tests: ./run_in_screen.sh (recommended)
   OR: ./run_comprehensive_test.sh (foreground)
   OR: See RUN_TESTS.md for custom tests

COMPREHENSIVE TEST
------------------
The included run_comprehensive_test.sh script runs:
- 2 models (gemini-3-flash-preview, gemini-2.5-flash)
- 4 request sizes (1k, 2k, 5k, 10k tokens)
- 12 thinking configs (4 levels + 8 budgets)
- 10 iterations per test
- ~96 total test combinations
- Estimated 2-4 hours
